# Fabric Cost Analysis

This folder contains a static HTML version of the Excel-based Fabric cost model.

## Files

- `index.html`: main calculator UI
- `styles.css`: visual styling
- `app.js`: pricing data and browser-side calculations

## Run locally

Open `index.html` directly in a browser, or serve the folder with a simple static web server.

## Notes

- The pricing table and scenario logic were ported from `../Fabric Cost Analysis.xlsx`.
- The first pass focuses on the main cost analysis workflow and a lightweight licensing comparison.
- It is designed to work well on a website or GitHub Pages. A GitHub README can link to it, but cannot host the interactive experience inline.
